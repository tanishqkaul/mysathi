# Project Assignment: Gamification UI Flow

## 🛠 Technical Requirements

To ensure a scalable and maintainable application, please adhere to the following guidelines:

*   **Framework & Libraries:** Use **React** for the application logic and **Tailwind CSS** (with **Shadcn UI** if required) for styling.
*   **State Management:** Use **Redux** to manage and store all application data.
*   **Database & Backend:** Use **Supabase** as the backend database. Leverage the **Supabase MCP** to design the database schema and create necessary tables to support the application's data needs.
